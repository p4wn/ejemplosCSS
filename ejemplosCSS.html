Ejemplos de CSS útiles

Truncado de texto:

Si se hace doble click se selecciona todo el texto, aunque no se puede recorrer

ejemplo

Texto que vamos a truncar y poner tres puntos a continuación bla bla bla bla bla bla

código 
width: 30%;
border: 2px solid;
white-space: nowrap;
overflow: hidden;
text-overflow: ellipsis;

Centrado de contenido:

Centrar un bloque o cualquier elemento al centro

ejemplo

Estos

textos

están centrados

código 
display: flex;
align-items: center;
flex-direction: column;
justify-content: center;

Primera letra grande

Aumentar la primera letra del párrafo a un tamaño superior al resto

ejemplo

Esta letra E debería ser mucho más grande que el resto

código 
p:first-child::first-letter{
    font-size: 200%;
    font-weight: bold;
}

Campos obligatorios formulario

Remarca en rojo los campos que son obligatorios en un formulario y los opcionales con marco negro

ejemplo
código 
HTML:
<div class="container-form-required">
    <label for="nombre">Nombre: <input type="text" name="nombre" id="nombre" required></label>
    <label for="apellidos">Apellidos: <input type="text" name="apellidos" id="apellidos"></label>
</div>

CSS:
#nombre:required {
    border: 2px solid red;
}

#apellidos:optional {
    border: 2px solid black;
}
